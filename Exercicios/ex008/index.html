Principais Formatações

Negrito / Destaque

Nesta frase um termo negrito utilizando a tag <b> (não semântica).

Nesta frase um termo em destaque usando a tag <STRONG> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântico)

Nesta frase temos um termo

Nesta frase temos um termo em ênfase usando a tag <em> (semântico)

Texto marcado

podemos criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Temos um texto grande e um texto pequeno

Texto Deletado

Podemos marcar um texto como excluido para indicar, que ele deve ser lido, mas não considerado, usa-se a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usa-se a tag <ins> (semântico).

Existe também o sublinhado com a tag <u> (não semântico).

Texto Sobrescrito

Para inserir coisas do tipo x20+3, use a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O, use a tag <sub>
