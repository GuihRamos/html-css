Título

<id>

Tag usada para darmos uma indentificação a uma determinada tag, para ficar mais facil de estilizarmos a tag, assim só estilizando a tag com o 'id'.

⇒OBS⇐

Só se deve utilizar um "id" APENAS UMA VEZ em uma PÁGINA, exemplo:
Se criarmos um id com nome "título principal", O indicado é que só utilizemos essa id uma única vez na página do documento, o mesmo vale para as outras id.

<Class>

Essa tag funciona parecida com o <id>, a diferença é que ela foi feita para utilizarmos mais de uma vez exemplo:
se em um documento tivermos 3 parágrafos destaque, podemos dar a eles uma classe, e assim só estilizamos o class e os 3 parágrafos vão possuir a mesma estilização.

Pseudo-Class

Utilizado para determinar o estado de um elemento, se ele está (ativo, se o mouse está passando por cima, marcado, vazio e etc).

Alguns Pseudo-Class Importantes

  • hover ⇒ Seleciona onde vc passar o mouse por cima ⇐
  • active ⇒ Seleciona onde vc está clicando
  • link ⇒ Seleciona o que já foi visitado (ex: links ancorados ao site) ⇐

Pseudo-Elements

Utilizado para especificar partes específicas de um elemento, como a primeira letra, a primeira linha, adicionar algo antes ou depois.

Alguns Pseudo-Elements Importantes

  • before ⇒ Adiciona algo antes do elemento ⇐
  • after ⇒ Adiciona algo após o elemento ⇐
  • first-line ⇒ Seleciona a primeira linha do elemento ⇐
